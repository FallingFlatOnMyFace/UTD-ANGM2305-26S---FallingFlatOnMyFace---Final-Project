Place image files (JPG or PNG format) in this folder in order for the program to read them.

Images must meet this criteria:

1. Black and white only
2. symmetrical proportions (no 800x1100 or likewise [Must be 500x500 or similar, 600x600, 700x700, etc.])
3. JPG or PNG only

An example image has been left in the folder for testing and reference.
